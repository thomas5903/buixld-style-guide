System Colors

System colors are used in standard areas such as side navigation and footers. System colors should be used when looking for a contrast within a given entity color scheme. The strategic use of system colors is strongly encouraged.

Entity Colors

Entity branding is a specific palette of colors associated with a given HCA brand. These colors will vary from brand to brand, and will act as the primary scheme for the site. In instances where entity branding may become too strong, we suggest using system colors as an offset.
